## 一、模块列表 ##
========
### BLOCK A ###
**BLOCKA 用于叠放1PIN模块** 

|  *  |模块名    |英文名  |分区 |默认引脚|
|:---:|:---:    |:---:  |:---:|:------:|
|1    |光敏电阻  |light  |A    | |
|2    |DS18B20  |DS18B20|D    | |
|3    |有源蜂鸣器|P_beep |C    | |
|4    |无源蜂鸣器|N_beep |C    | |
|5    |555      |555    |B    | |

因为蜂鸣器一般一次只选用一款
所以我们的设定是有源和无源蜂鸣器都占用BLOCK的同一个I/O
如果有需要同时用到两种蜂鸣器就只能放一个在BLOCKB了
